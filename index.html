About Me

My background is not limited to one narrow cybersecurity lane. I have worked in real enterprise environments where systems had to stay online, vulnerabilities had to be remediated, databases had to be maintained, access had to be controlled, and mission-critical healthcare applications had to remain secure and available.

I bring a hybrid skill set across cybersecurity operations, systems administration, Identity and Access Management, SQL Server administration, data analysis, PKI, vulnerability management, compliance, and application support. That means I understand both sides of the work: the security controls that protect the environment and the infrastructure those controls depend on.

I have supported Department of Defense healthcare environments involving Windows Server, VMware, SQL Server, Active Directory, IIS, Citrix, RDS, Microsoft Defender, Nessus/ACAS, Trellix SIEM, RMF, NIST 800-53, DISA STIGs, eMASS, POA&M tracking, HIPAA requirements, PKI certificates, CAC authentication, and secure enterprise applications.

This portfolio exists to show proof of work. The labs below demonstrate hands-on capability across enterprise security, identity governance, Zero Trust IAM, PII compliance, access control, automation, and defensive security operations.
